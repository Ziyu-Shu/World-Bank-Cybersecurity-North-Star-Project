# World-Bank-Cybersecurity-North-Star-Project
This is my personal project as the practicum of my master program in collaboration with the World Bank Digital Development Team. 

## Project Overview

This project will create an interactive world map that visualizes cybersecurity capacity and maturity across approximately 50 developing countries. Users will be able to hover over countries to see indicator-level data on legal frameworks, technical capacity, organizational measures, workforce readiness, and international cooperation. The tool will serve as a "North Star" for World Bank staff advising governments on where to invest in cybersecurity capacity-building.

Data has already been collected for approximately 50 countries and is being refined. Students will build the interactive scaffolding and visualization layer, designed to be updatable as the underlying data matures.

## Background & Context
Cyber incidents in developing countries are rising 37% annually in upper-middle-income countries and 21% worldwide, with costs reaching up to 3% of GDP in some nations. The World Bank has supported 64 countries on cybersecurity since 2013 through its Cybersecurity Multi-Donor Trust Fund, backed by Estonia, Israel, Japan, the Netherlands, the USA, and the Bill & Melinda Gates Foundation.

Despite this investment, critical gaps remain. Only 5 of 22 Western/Central African countries have operational Computer Security Incident Response Teams (CSIRTs), and only 10 of 26 Eastern/Southern African countries do. The ITU Global Cybersecurity Index (GCI) assesses nearly 200 economies across five pillars, while the Cybersecurity Capacity Maturity Model (CMM) has been deployed in 85+ countries. This project will synthesize these assessments into a single, actionable visual tool.

## Key Questions for Student Investigation
What are the most significant cybersecurity capacity gaps across the 50 target countries, and do they cluster by region or income level?

How do countries compare on legal/regulatory frameworks vs. technical/operational capacity, and what does that mismatch imply for investment?

Which indicator dimensions show the strongest correlation with reduced cyber incident frequency or economic impact?

How can the interactive map effectively communicate both current state and trajectory (improvement over time) to non-technical policymakers?
   
What are the best practices from top-performing developing countries (e.g., Rwanda, Ghana, Singapore) that could be highlighted as models?
